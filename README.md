# Cartelera Effi

Cartelera digital de Grupo Effi. SPA en React + TypeScript + Vite con Tailwind, shadcn/ui y Supabase.

## Stack
- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (datos)
- Deploy en Vercel

## Desarrollo local

```sh
npm install
npm run dev
```

La app queda disponible en `http://localhost:8080`.

## Build de producción

```sh
npm run build
npm run preview
```

## Deploy
El proyecto se despliega automáticamente a Vercel al hacer push a `main`.
